Getting started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

Prerequisites

If you don't have Nodejs and npm installed, install them from here.. You'll also want the Vue CLI tool, which you can install like this:

npm i -g @vue/cli

Installing and Running

Clone this repository to your local machine, and in the root directory:

Install Dependencies

npm i

npm run lerna-install

Available Client Commands

Compiles and hot-reloads for development

npm run serve

Compiles and minifies for production

npm run build

Run your unit tests

npm run test:unit

Run your end-to-end tests

npm run test:e2e

Lints and fixes files

npm run lint

Customize configuration

See Configuration Reference.

Available Server Commands

Hot reload for development

npm run dev

Build the server

npm run build

Start the server

npm run start
